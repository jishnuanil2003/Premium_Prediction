# 🏥 Insurance Premium Prediction using Machine Learning

## 📌 Project Overview
This project predicts **insurance premium costs** based on user details such as age, BMI, smoking habits, and other health/lifestyle factors.

The goal is to build a robust machine learning model that helps insurance companies estimate premium prices more accurately.

---

## 🎯 Problem Statement
Insurance companies need to determine premium costs based on multiple risk factors.  
Manual estimation is inefficient and error-prone.

👉 This project solves that using **Machine Learning regression models**.

---

## 📊 Dataset Features
- Age  
- Gender  
- BMI (Body Mass Index)  
- Number of Children  
- Smoking Status  
- Region
